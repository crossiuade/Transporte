# Transportes de Tigre

Sitio web simple y responsive para una propuesta de transporte en Tigre con cinco servicios disponibles.

## Estructura del proyecto

- `index.html` - Página principal con enlaces a cada transporte.
- `transporte1.html` - Página de Transporte 1.
- `transporte2.html` - Página de Transporte 2.
- `transporte3.html` - Página de Transporte 3.
- `transporte4.html` - Página de Transporte 4.
- `transporte5.html` - Página de Transporte 5.
- `contacto.html` - Página de contacto con formulario.
- `Estilo/styles.css` - Hoja de estilos compartida en tonos blancos y azules.
- `Img/` - Imágenes SVG usadas en las páginas.

## Cómo usar

1. Abre `index.html` en tu navegador.
2. Navega entre las páginas con los enlaces del menú.
3. En `contacto.html` puedes completar el formulario para simular una consulta.

## Notas

- No usa frameworks.
- Es responsive y adaptado a pantallas móviles.
- El formulario usa `method="post"` y `action="#"` como ejemplo.
